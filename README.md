## 🌟 About

StudyMate AI is a modern web application that helps students study smarter using artificial intelligence. Upload your notes, get AI-powered summaries, ask questions about your documents, and track your progress.

## ✨ Features (Planned)

- 📚 AI-powered document summarization
- 🤖 Chat with your study materials
- ⏱️ Pomodoro timer with statistics
- 📊 Progress tracking and analytics
- 🎯 Smart task management
- 💡 AI-generated flashcards
- 📅 Personalized study planner

## 🛠️ Tech Stack

**Frontend:**
- React 18 + TypeScript
- TailwindCSS
- Vite

**Backend:**
- Python 3.11+
- FastAPI
- PostgreSQL
- SQLAlchemy

**AI/ML:**
- OpenAI GPT-4
- LangChain
- ChromaDB

**DevOps:**
- Docker
- GitHub Actions
- Vercel (Frontend)
- Railway (Backend)
